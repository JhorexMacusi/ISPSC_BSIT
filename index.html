How to Contribute

  1. Fork this repository on GitHub.
  2. Clone your forked repository to your computer.
  3. Add your portfolio or project inside the appropriate folder.
  4. Commit your changes with a clear message (e.g., Add my portfolio: Jhorex).
  5. Push your changes to your forked repo.
  6. Submit a Pull Request and wait for review.
